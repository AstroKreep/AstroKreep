🧬 Flagellum AI

A biologically-inspired NPC architecture modeled after the structure and behavior of a bacterial flagellum.

The system explores modular perception, environmental sensing, decision-making, directional control, locomotion, adaptation, and emergent NPC behavior.

⸻

🛡️ Rust Plugin Ecosystem

A growing collection of custom Rust systems, including:

* Dynamic server events
* NPC behavior tools
* Immersive vehicle mechanics
* Environmental hazards
* Player interaction systems
* Security and moderation tools
* Advanced control-center interfaces
* Server diagnostics and monitoring

Every plugin is built with configurability, performance, maintainability, and server-owner usability in mind.
