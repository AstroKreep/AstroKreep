> FEATURED_SYSTEMS

🧬 Flagellum AI

A biologically-inspired NPC architecture modeled after the structure and behavior of a bacterial flagellum.

The system explores:

* Modular perception
* Environmental sensing
* Decision-making
* Directional control
* Locomotion
* Adaptation
* Chemotaxis-inspired navigation
* Emergent NPC behavior

The goal is to create NPCs that react to their environment instead of simply following rigid scripts.
